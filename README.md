# 🐾 **전국 반려동물 등록·입양·유기 패턴 분석 및 예측 — 프로젝트 요약본**

## 📌 프로젝트 개요

본 프로젝트는 농림축산검역본부·국가동물보호정보시스템 Open API를 활용하여 **전국 반려동물 등록 현황, 유기·입양 추이, 보호소 운영 구조를 통합적으로 분석**하는 빅데이터 기반 연구이다. 대규모 유기동물(개·고양이 중심) 데이터와 전국 보호소 위치 정보를 결합하여, 시간·지역·동물 특성에 따른 패턴을 정밀하게 탐색하고 예측 모델을 구축하는 것을 목표로 한다.

---

## 📂 데이터 수집·저장

* 국가동물보호정보시스템 / 검역본부 / 보호소 표준데이터 Open API를 이용해 전국 단위 데이터를 자동 수집
* 유기동물, 등록현황, 보호소 정보를 **MongoDB 컬렉션으로 구조화**하여 효율적으로 관리
* 원본 JSON 형식을 유지하며, 월 단위·지역 단위로 안정적인 ETL 파이프라인 구축

---

## 🔧 데이터 가공·정제

* Pandas로 날짜, 지역, 중성화, 성별, 품종 등 주요 특성을 정제
* 계절(season), 요일(weekday), 나이(age), 지역코드 등 **파생 변수 자동 생성**
* 이상치·결측치 처리 후 분석 가능한 형태의 통합 데이터셋 구축

---

## 📊 분석 주요 결과

* **계절 패턴:** 여름철(5~8월) 유기 발생이 전국적으로 급증
* **지역 패턴:** 광주·서울·울산 등은 등록 대비 유기율이 높아 집중 관리 필요
* **처리결과 분석:** 입양 비율은 낮고, 안락사·자연사가 전체의 절반 이상을 차지
* **종·특성 분석:** 유기동물은 개·고양이에 압도적으로 집중, 성별·중성화 여부·나이와 높은 상관성
* **보호소 인프라:** 수도권·부산·경남에 보호소가 밀집, 강원·충북·전북 등은 접근성 취약

---

## 🤖 머신러닝 기반 예측

* RandomForest 등 모델을 활용해 **유기 발생·처리결과 예측 모델** 구축
* 지역·계절·개체 특성을 반영한 예측 결과를 통해 정책 설계 가능성 확인

---

## 📈 데이터 시각화

* 시계열 그래프, Heatmap, 등록 대비 유기율 비교 차트 등 다각적 시각화
* folium 기반 **전국 보호소 위치 지도(HTML)** 제작
* Tableau용 집계데이터 생성으로 대시보드 확장 가능

---

## 🧩 프로젝트 의의

본 연구는 전국 단위의 공공데이터를 통합 분석하여 **유기동물 발생 원인, 지역 격차, 보호소 운영 문제, 입양 정체의 구조적 요인**을 객관적으로 도출하였다. 이러한 분석 결과는 보호소 자원 재배치, 계절별 집중 관리, 등록제 강화, 유기 방지 캠페인 등 **정책적 개선 방향을 제시할 수 있는 중요한 근거**가 된다. 또한 대용량 데이터 기반의 예측 모델을 통해 향후 유기 발생 위험과 입양률 변화를 과학적으로 전망할 수 있는 점에서 실무적 가치가 크다.
